Our journey

Application développée en mobile first permettant de créer et sauvegarder ses carnets de voyages ainsi que ses astuces.

J'ai utilisé PHP et son framework Symfony ; Twig ; Bootstrap ; CSS3 ; HTML5 ; Javascript Vanilla ; Librairie Leaflet et l'API de geocoding de Open weather map pour les cartes ; MySQL et Adminer ; SQL.

D'un point de vue sécurité, j'ai mis en place : Contrôle des accès ; Algorithme de hachage ; Requêtes préparées ; JWT et CSRF token ; Vérifications des champs des formulaires ; CORS.
